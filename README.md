# ⚡ VibeFlow — Premium AI Productivity Suite

VibeFlow is a state-of-the-art, glassmorphic productivity assistant designed to streamline your workflow with integrated AI features, task management, document analysis, and smart note-taking.

![VibeFlow Banner](https://raw.githubusercontent.com/lucide-react/lucide/main/icons/zap.svg)

## 🚀 Key Features

- **📊 Dashboard Overview**: Real-time stats and insights into your productivity.
- **📋 Smart Task Manager**: Organize your daily work with priority tracking, due dates, and status filters.
- **📄 Document Intelligence**: Upload text documents to get instant AI-generated summaries and automated task extraction.
- **📒 Intelligent Notes**: Capture your thoughts with automatic summarization to help you focus on what matters.
- **🧠 Knowledge Hub**: A searchable repository that consolidates all insights extracted from your documents and notes.
- **🤖 VibeFlow Chat Assistant**: A context-aware, rule-based chatbot with a 100+ entry knowledge base to answer your productivity questions.
- **👤 User Profile**: Personalized account management with productivity stats and bio.

## 🛠️ Technology Stack

- **Frontend**: React 18, Vite (for blazing fast development and builds).
- **Styling**: Tailwind CSS for a custom, premium glassmorphic UI.
- **Animations**: Framer Motion for smooth, cinematic transitions.
- **Icons**: Lucide React for clean, consistent iconography.
- **Backend (BaaS)**: Firebase Authentication and Cloud Firestore for secure data storage.

## 📦 Getting Started

### Prerequisites

- Node.js (v16.x or later)
- npm or yarn

### Installation

1. **Clone the repository**:
   ```bash
   git clone <your-repository-url>
   cd VibeFlow
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set Up Firebase (Optional for local dev)**:
   The application is currently pre-configured with a demo Firebase project. For production use, please update `src/config/firebase.js` with your own project credentials.

4. **Run the development server**:
   ```bash
   npm run dev
   ```

The application will be available at [http://localhost:5000/](http://localhost:5000/).

## 🎨 Design Philosophy: "Cyber Glassmorphism"

VibeFlow uses a "Cyber Glassmorphic" design language, featuring:
- Semi-transparent, blurred layers.
- High-contrast, vibrant accents on deep backgrounds.
- Premium micro-animations for interactive elements.
- Clean, modern typography (Inter/Roobto fallback).
